# MedicalPortal (Backend)
The backend for the MedicalPortal system.


# Installation

### Libraries / Packages
```
pip install flask
pip install flask-cors
pip install mysql-connector-python
```

# Running App
Execute in the `/backend/python` folder
```
flask --app backend run
```
